# Документация

Этот каталог разделён на 2 зоны:

- `specs/` — живая техническая документация, которая описывает текущие контракты и поведение.
- `archive/` — исторические планы, дорожные карты и старые рабочие документы. Они полезны как контекст, но не являются source of truth для текущей реализации.

## Актуальные документы

- [`specs/evidence-anchor-spec-v0.md`](./specs/evidence-anchor-spec-v0.md) — спецификация evidence/anchor для перехода из чата к источнику.

## Архив

- [`archive/plans/`](./archive/plans/) — старые планы развития, UX-полировки и поэтапной реализации.
- [`archive/bundles/offline-amd64-bundle.txt`](./archive/bundles/offline-amd64-bundle.txt) — инструкция к офлайн bundle под amd64.

Если нужно понять, как запускать и использовать систему сейчас, начинайте с корневого [`README.md`](../README.md).
